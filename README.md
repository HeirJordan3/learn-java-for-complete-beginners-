# ☕ Learn Java for Complete Beginners

A step-by-step, beginner-friendly course designed to teach you Java from zero.

Whether you’ve never written code before or you’re looking to strengthen your fundamentals, this course walks you through Java in a clear, structured way.

---

## 🎯 Who This Course Is For

* Complete beginners
* High school students
* Career changers
* Self-taught learners who want strong fundamentals
* Anyone curious about programming

No prior experience required.

---

## 📚 What You’ll Learn

By the end of this course, you will be able to:

* Understand how programming works
* Write and run Java programs
* Use variables and data types
* Work with conditionals and loops
* Create and use methods
* Understand arrays and ArrayLists
* Learn the foundations of Object-Oriented Programming
* Build a small final project

---

## 🗂 Course Structure

| Day           | Topic                         |
| ------------- | ----------------------------- |
| Day 1         | Introduction to Java          |
| Day 2         | Variables & Data Types        |
| Day 3         | Conditionals                  |
| Day 4         | Loops                         |
| Day 5         | Methods                       |
| Day 6         | Arrays                        |
| Day 7         | ArrayLists                    |
| Day 8         | Object-Oriented Programming   |
| Final Project | Build a Mini Java Application |

Each day includes:

* 📖 Lesson explanation (markdown)
* 💻 Example code
* 🧪 Practice exercises
* ✅ Solutions

---

## 🚀 How To Use This Course

1. Clone this repository.
2. Complete the setup in the `setup/` folder.
3. Start with `Day-01-Intro-To-Java`.
4. Read the lesson.
5. Run the example code.
6. Complete the exercises before checking solutions.

Move at your own pace.

---

## 🛠 Requirements

You will need:

* Java JDK (Version 17+ recommended)
* IntelliJ IDEA (Community Edition recommended)
* A computer (Mac, Windows, or Linux)

---

## 🧠 Why Java?

Java is one of the most widely used programming languages in the world.

It is used in:

* Enterprise software
* Banking systems
* Android applications
* Backend development
* Large-scale systems

Learning Java builds strong programming fundamentals that transfer to other languages.

---

## 🏁 Final Project

At the end of this course, you will build a small console-based Java application that combines everything you’ve learned.

This helps reinforce real-world problem solving and confidence.
